# Вывод

Тестовая программа "Hello, world" выглядит так:
```lua
func main()
    println("Hello, world!!!")
end
```

Пояснение:
```
func main()                  входная точка программы
println("Hello, world!!!")   вывод текста Hello, world!!! на экран
end                          конец функции
```

**все тестовые программы на языке Iron есть в папке examples!**
